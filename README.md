# Bus Ticket System

Sistem pemesanan tiket bus online berbasis web dengan PHP Native MVC.

## Fitur Utama

1. **Pemesanan Tiket Online**
   - Pencarian jadwal berdasarkan rute dan tanggal
   - Pemilihan kursi interaktif
   - Multi-penumpang dalam satu pemesanan
   - Berbagai metode pembayaran

2. **Manajemen Pengguna**
   - Registrasi dan login
   - Dashboard pengguna
   - Riwayat pemesanan
   - Manajemen profil

3. **Panel Admin**
   - Kelola bus, rute, dan jadwal
   - Kelola pemesanan dan pembayaran
   - Laporan dan statistik
   - Manajemen pengguna

4. **Keamanan**
   - Autentikasi pengguna
   - CSRF protection
   - Input validation dan sanitization
   - Session management

## Teknologi

- **Backend**: PHP 7.4+ (Native MVC)
- **Database**: MySQL
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **CSS Framework**: Bootstrap 5
- **Icons**: Bootstrap Icons
